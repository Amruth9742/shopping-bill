# Shopping Cart Billing App

## Tech Stack
- React + TypeScript
- Redux Toolkit
- Tailwind CSS

## Features
- Add/remove products
- Apply special offers
- Calculate subtotal, savings, final total

## Offers
- Buy 1 Get 1 Cheese
- Soup + Bread discount
- Butter 1/3 off

## Run
npm install
npm run dev
